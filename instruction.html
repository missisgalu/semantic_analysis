Добро пожаловать в SA!

Приложение предоставляет удобный способ для поиска вакансий и анализа семантической близости между текстами. Вы можете искать вакансии по ключевым словам, исключать нежелательные слова, а также сохранять результаты поиска в удобном формате.

Мы также предоставляем инструменты для анализа семантической близости между текстами. Вы можете загрузить модели Word2Vec для вакансий и дисциплин, сравнить их и получить информацию о сходстве слов.

Инструкция по использованию приложения

    Шаг 1: Установка приложения

    После завершения загрузки, найдите загруженный архив проектом на вашем компьютере.

    Щелкните правой кнопкой мыши на архиве и выберите "Извлечь все" или аналогичную опцию для распаковки архива.

    Выберите место на вашем компьютере, где вы хотите сохранить разархивированные файлы билда.

    Шаг 2: Запуск приложения

    Перейдите в папку, где вы разархивировали билд проекта.

    Найдите исполняемый файл приложения (.exe) и дважды щелкните по нему, чтобы запустить приложение.

    Шаги на скриншотах

    Шаг 3: Авторизация

    Запустите приложение и нажмите кнопку авторизации.

    Введите на сайте hh.ru свои учетные данные и авторизируйтесь

    Нажмите "Продолжить" при переходе на страницу "Использовать приложение «Парсер вакансий для диплома»"

    Найдите токен доступа (символы в ссылке после https://example.com/vacancy-parser?code=)

    Скопируйте символы в приложение

    Нажмите "Ввести код"

    Шаг 4: Поиск вакансий

    На вкладке "Поиск вакансий" введите ключевые слова для поиска вакансий.

    Вы можете также указать слова, которые следует исключить из результатов.

    Нажмите кнопку "Найти вакансии" для выполнения поиска.

    По желанию можно сохранить результаты поиска в таблицу Excel.

    Нажмите кнопку "Обучить модель" и сохраните результаты для дальнейшего сравнения

    Шаг 5: Обучение модели дисциплин

    Для анализа и классификации дисциплин вы можете обучить модель Word2Vec на текстах, связанных с этими дисциплинами.

    1. Выберите папку, в которой содержатся тексты дисциплинных материалов.

    2. Нажмите кнопку "Выбрать папку с дисциплинами" и укажите путь к выбранной папке.

    3. Нажмите кнопку "Обучить модель" для запуска процесса обучения модели Word2Vec.

    4. После завершения обучения модели вы сможете просмотреть список слов, включенных в модель, и сохранить обученную модель для дальнейшего использования.

    Обучение модели позволит вам проводить семантический анализ и классификацию текстовых документов, связанных с дисциплинами, и получать более точные результаты при поиске похожих материалов.

    Шаг 6: Сравнение моделей

    Загрузка модели вакансий:

    • Нажмите кнопку "Загрузить модель вакансий".
    • Выберите файл модели вакансий в диалоговом окне.
    • Модель вакансий будет загружена, и информация о загруженной модели будет отображена.

    Загрузка модели дисциплин:

    • Нажмите кнопку "Загрузить модель дисциплин".
    • Выберите файл модели дисциплин в диалоговом окне.
    • Модель дисциплин будет загружена, и информация о загруженной модели будет отображена.

    Нажмите кнопку "Сравнить модели".

    Результаты сравнения будут представлены в таблице.

    Будут вычислены и отображены метрики сходства между случайно выбранными словами из моделей.

    Шаг 7: Составление графиков

    Нажмите кнопку "Составить графики".

    Графики сравнения слов между моделями строятся и отображаются.

    Также можно соханить график
